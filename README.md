# Modern Store - Altyapı ve Veritabanı (Persistence Layer)

Projemin verilerle tıkır tıkır "konuştuğu" yer tam olarak burası. Core katmanında oluşturduğum tüm yapıları, burada gerçek bir veritabanına bağlayarak hayat verdim.

### Bu Dalda Neler Yaptım?
*   **Core Altyapısı:** Projenin kalbi olan Domain ve Application katmanlarını temel aldım.
*   **Veritabanı Bağlantısı:** Entity Framework Core kullanarak veritabanı tablolarımı ve ilişkilerimi kurguladım.
*   **Repository Pattern:** Veri işlemlerimi daha profesyonel ve yönetilebilir hale getirmek için Repository desenini uyguladım.
*   **Migrations:** Veritabanı şemamdaki tüm değişiklikleri migration dosyalarıyla adım adım takip ettim.

Açıkçası bu aşamada uygulamam artık verileri kalıcı olarak saklayabilen ve yönetebilen profesyonel bir altyapıya kavuşmuş oldu.
